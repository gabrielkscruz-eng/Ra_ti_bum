# 🎮 Ra-ti-bum

Projeto educacional de engenharia da computação - Sistema interativo de automação com cenário temático, sensores, servos e Arduino.

## 📋 Sobre o Projeto

Ra-ti-bum é um projeto integrador desenvolvido na Universidade Santa Cecília. O projeto combina eletrônica, programação e modelagem 3D para criar um sistema automatizado com tema de assustamento (Dia das Bruxas).

O sistema utiliza sensores de luz (LDR), botões, relés, servos e um Arduino para controlar diversos mecanismos como um elevador, portas, janelas e elementos decorativos em uma maquete temática.

## 🛠️ Componentes Utilizados

### Hardware
- **Arduino** - Microcontrolador principal
- **4x Micro Servo** - Controle de movimentos
- **4x LDR (Light Dependent Resistor)** - Sensores de luz
- **2x Push Button** - Botões de controle
- **2x Relé** - Chaveamento de circuitos
- **2x Transistor** - Amplificação de sinal
- **2x Diodo** - Proteção de circuitos
- **Motor DC** - Elevador (5V, 50mA operacional, 100mA pico)
- **Buzzer** - Efeitos sonoros
- **2x Botão Gangorra** - Interação do usuário
- **Resistores, Capacitores e Fios Jumper**

### Software
- **C++** - Programação do Arduino
- **Tinkercad** - Planejamento inicial do circuito
- **KiCAD** - Design do circuito final
- **Fusion 360** - Modelagem 3D do cenário
- **Arduino IDE** - Programação e upload

### Fabricação
- **Cortadora a Laser** - Construção do cenário
- **Impressora 3D** - Peças decorativas e polias
- **Prototipadora** - Fabricação da placa PCB

## 📱 Funcionalidades

- Elevador automatizado com controle de servo
- Portas e janelas com movimento controlado
- Sensores de luz para detecção de presença
- Sistema de som com buzzer
- Efeitos de iluminação via LDR
- Controle via botões interativos

## 🔧 Como Funciona

1. **Sensores LDR** detectam mudanças de iluminação no ambiente
2. **Botões** permitem controle manual do usuário
3. **Arduino** processa os sinais e controla os servos
4. **Servos** acionam movimentos mecânicos (elevador, portas, janelas)
5. **Buzzer** emite sons para efeitos especiais
6. **Motor DC** movimenta o elevador
